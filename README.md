
# 5G-NR-V2X 

[5G-NR-V2X] 
 1. KIAPI OBU 연동 소프트웨어 KETI 군집주행 서비스 메시지 프로토콜 생성 및 처리 구현 코드
    -


  - 본 프로그램은 5G-NR-V2X 통신을 이용하여 LV, FV 메시지를 교환하여 군집주행 서비스 처리
  - (LV -> FV 메시지 수신 처리)
  - (FV -> LV 메시지 전송 처리)

2. 프로그램 활용 시 참고사항
   -

- 해당 프로그램 코드의 경우, 플래투닝 서비스를 위한 후행차량(Follow Vehicle)을 대상으로 LV 메시지를 수신하여 제공하는 코드 일부를 포함
- FV 메시지를 생성하여 LV 차량으로 정보를 제공하기 위한 메시지 생성 코드부를 포함
- 따라서, 5G-NR-V2X 장치 간 정보 연계 코드는 포함되어 있지 않으며, 일반적인 유틸리티(TCP 핸들링, 값 변환 등) 코드는 포함되어 있지 않습니다.
- 본 코드를 Build 하기위해서는 2가지의 추가적인 라이브러리(libnr-v2x, libcpp-framework) 코드가 필요하며 필요 시 개발자의 연락처로 요청주시면됩니다.
